Data Reporting Class Story Pitch

Jessica Baugh

2025-10-16

Hundreds of unaccompanied migrant children go to a rural Washington County, and its classrooms.

A rural Washington county has one of the state’s highest numbers of unaccompanied migrant children, trailing only Seattle-area counties. Seventy percent of these children are not going to their parents. 

Mason County received 412 unaccompanied children from January 2015 through May 2023, according to data from the U.S. Department of Health and Human Services. This is more than triple the county average in Washington state. Of these children, 60% were placed with non-parent sponsors. 

Since 2015, there has been a similar number of migrant children arriving. As they arrive and adapt to living in Mason County, school can be a place of support and structure. Do the local schools in Mason County have programs to help this population of students settle? 

This reporting will build on The New York Times’ investigation “Where Migrant Children Are Living, and Often Working, in the U.S.,” which marked Mason County as the hotspot of Washington for non-parent sponsor placements. Exploring how or if school districts address this population of children can show how local policies and organizations have adapted to account for the number of migrant children coming to Mason County. 

Setup

# Loading necessary packages 
pacman::p_load(tidyverse, lubridate, janitor, tidycensus, zipcodeR, stringr)

# Loading in the NYTimes dataset from GitHub
nyt_mc <- read_csv("data.csv", col_types = cols(ID = col_character(), 
    `Child's Date of Entry` = col_date(format = "%m/%d/%y"), 
    `Child's Date of Release` = col_date(format = "%m/%d/%y"), 
    `Sponsor Category` = col_character()))
nyt_mc <- clean_names(nyt_mc)

# Selecting the Mason County zipcodes
mason_zips <- c("98524","98528","98546","98548","98555","98560","98584","98588","98592")

mason_mc <- nyt_mc %>%
  mutate(
    sponsor_zip_5 = str_extract(as.character(sponsor_zipcode), "^\\d{5}")
  ) %>%
  filter(sponsor_zip_5 %in% mason_zips)

When migrant children come to Mason County, 60% of them do not go to parent sponsors. Most go to immediate relatives, such as a sibling, aunt, uncle or cousin.

Getting + joining zipcode data

# Getting zipcode data for WA
zip_to_county_WA <- zipcodeR::zip_code_db %>%
  filter(state == "WA") %>%
  transmute(
    sponsor_zipcode = zipcode,
    county   = county,    
  )

# Joining data and narrowing it down
wa_nonparent <- nyt_mc %>%
  left_join(zip_to_county_WA, by = "sponsor_zipcode") %>%
  filter(
    !is.na(county),              
    sponsor_category %in% c("2","3")
  )

Assigning zipcodes to counties

# Build WA county FIPS from names
norm_county <- function(x) {
  x |>
    stringr::str_to_lower() |>
    stringr::str_replace_all("\\bcounty\\b", "") |>
    stringr::str_replace_all("[^a-z ]", " ") |>
    stringr::str_squish()
}

wa_fips <- tidycensus::fips_codes %>%
  filter(state == "WA") %>%
  transmute(
    county_name_clean = norm_county(county),               # "mason"
    county_fips       = paste0(state_code, county_code)    # e.g., "53045"
  ) %>%
  distinct()

wa_counts <- wa_nonparent %>%
  mutate(county_name_clean = norm_county(county)) %>%
  inner_join(wa_fips, by = "county_name_clean") %>%
  count(county_fips, name = "children_nonparent")

Joining counts with shapes/population + finding rate

wa_rates <- wa_county_map %>%
  mutate(GEOID = as.character(GEOID)) %>%
  transmute(
    county_fips = stringr::str_pad(GEOID, width = 5, pad = "0"),
    county_name = NAME,
    population  = estimate,
    geometry
  ) %>%
  left_join(wa_counts, by = "county_fips") %>%
  mutate(
    children_nonparent = tidyr::replace_na(children_nonparent, 0L),
    rate_per_100k      = 100000 * children_nonparent / population
  )
